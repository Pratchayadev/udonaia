# AIA Insurance Advisor Website Plan
Version 1.0  
Purpose: Lead Generation + Personal Branding + Conversion Optimization

# 1. Website Strategy

## Primary Goal
- เก็บ Lead (ชื่อ + เบอร์ + Line)
- สร้างความน่าเชื่อถือ
- ปิดการขายผ่านการนัดคุย

## Target Audience
- อายุ 25–45 ปี
- มนุษย์เงินเดือน / เจ้าของธุรกิจขนาดเล็ก
- มีรายได้ 20,000–100,000 บาท/เดือน
- เริ่มกังวลเรื่องค่ารักษาพยาบาล / ครอบครัว / ภาษี

## Core Message
"วางแผนประกันให้เหมาะกับชีวิตคุณ ไม่ขายเกินจำเป็น"

# 7. SEO Blog Structure

บทความตัวอย่าง:

- ประกันสุขภาพเหมาจ่ายคืออะไร
- AIA ดีไหม
- ประกันเด็กจำเป็นไหม
- รายได้ 30,000 ควรทำประกันแบบไหนดี
- ค่ารักษาโรคมะเร็งปีล่าสุด

SEO Strategy:
- 2 บทความต่อสัปดาห์
- 800–1500 คำต่อบทความ
- ใส่ CTA ทุกบทความ
